# 一键删除无效参照 - 使用指南

## 📋 功能概述

这是一个AutoCAD LSP脚本，可以**一键删除所有无效的参照对象**，帮助您快速清理和优化CAD图纸。

### 支持清理的对象类型：
- ✓ 无效的外部参照(Xref)
- ✓ 无效的块参照(Block Reference)
- ✓ 无效的光栅图像(Raster Image)
- ✓ 无效的照片和OLE对象
- ✓ 无效的表格(Table)
- ✓ 无效的图层(Layer)
- ✓ 其他无效的几何对象
- ✓ 无效的应用程序注册表(RegApps)
- ✓ 无效的字典对象(包括图片字典)

---

## 🚀 快速开始

### 第一步：加载脚本

**方法一：使用APPLOAD命令**
1. 在AutoCAD命令行输入：`APPLOAD`
2. 在弹出的对话框中选择 `delete-invalid-refs.lsp` 文件
3. 点击"加载"按钮

**方法二：直接在命令行加载**
