# 一键删除无效参照 - 使用指南

## 📋 功能概述

这是一个AutoCAD LSP脚本，可以**一键删除所有无效的参照对象**，帮助您快速清理和优化CAD图纸。

### 支持清理的对象类型：
- ✓ 无效的外部参照(Xref)
- ✓ 无效的块参照(Block Reference)
- ✓ 无效的光栅图像(Raster Image)
- ✓ 无效的照片和OLE对象
- ✓ 无效的表格(Table)
- ✓ 无效的图层(Layer)
- ✓ 其他无效的几何对象
- ✓ 无效的应用程序注册表(RegApps)
- ✓ 无效的字典对象(包括图片字典)

---

## 🚀 快速开始

### 第一步：加载脚本

**方法一：使用APPLOAD命令**
1. 在AutoCAD命令行输入：`APPLOAD`
2. 在弹出的对话框中选择 `delete-invalid-refs.lsp` 文件
3. 点击"加载"按钮

**方法二：直接在命令行加载**
```
(load "delete-invalid-refs.lsp")
```

**方法三：放入启动文件夹（自动加载）**
1. 将脚本放入AutoCAD启动文件夹
2. 重启AutoCAD，脚本会自动加载

### 第二步：执行清理命令

在AutoCAD命令行输入以下任意一个命令即可：
```
CLEANINVALID
```
或简写：
```
CI
```

### 第三步：查看清理结果

脚本会自动：
1. 扫描图纸中的所有无效对象
2. 逐一删除无效参照
3. 显示详细的清理报告

## ⚙️ 主要功能详解

### 1️⃣ 外部参照清理 [1/9]
- 检测并删除无效或已断开的外部参照(Xref)
- 保留有效的外部参照

### 2️⃣ 块参照清理 [2/9]
- 删除指向无效块定义的块参照
- 清理损坏的块参照对象

### 3️⃣ 光栅图像清理 [3/9]
- 删除源文件不存在的光栅图像
- 清理损坏的图像对象和Wipeout对象
- **这是最常见的清理需求！**

### 4️⃣ 表格清理 [4/9]
- 检测并删除损坏的Table对象
- 清理格式错误的表格

### 5️⃣ OLE对象清理 [5/9]
- 删除无效的OLE嵌入对象
- 清理失效的照片和文档链接

### 6️⃣ 图层清理 [6/9]
- 删除未使用的图层
- **自动保护系统图层**（0、DEFPOINTS、XREF_DEFS）
- 防止意外删除正在使用的图层

### 7️⃣ 其他对象清理 [7/9]
- 删除损坏的多边形和线条对象
- 清理格式错误的几何对象

### 8️⃣ RegApps清理 [8/9]
- 清理无效的应用程序注册表
- 移除已删除程序的残留数据

### 9️⃣ 字典清理 [9/9]
- 清理图片字典中的无效条目
- 删除损坏的命名对象字典
- **自动保护系统必需的字典**

## ⚠️ 重要提示

### ✅ 安全特性
- ✓ **自动撤销保护** - 清理失败可用 `Ctrl+Z` 撤销
- ✓ **系统保护** - 不会删除必要的系统图层和字典
- ✓ **错误处理** - 删除失败会显示警告，继续处理其他对象

### ❌ 使用前注意

1. **备份重要图纸**
   ```
   先保存一份备份副本再进行清理
   ```

2. **关闭不必要的参照**
   - 建议先卸载(Unload)无用的外部参照
   - 关闭不需要的图像链接

3. **不要在图纸被编辑时运行**
   - 确保没有其他程序或用户正在编辑此图纸

## 🔍 常见问题解答

### Q1: 命令加载后没有反应？
**A:** 
- 检查文件名是否正确：`delete-invalid-refs.lsp`
- 确保AutoCAD命令行中显示"已加载"的提示
- 尝试重新加载：`APPLOAD` → 重新选择文件

### Q2: 执行命令后卡住了？
**A:**
- 这是正常的，取决于图纸复杂度
- 大型图纸可能需要几分钟
- **不要强制关闭，耐心等待**

### Q3: 删除了不该删除的对象？
**A:**
- 立即按 `Ctrl+Z` 撤销
- 脚本支持完整的撤销功能

### Q4: 光栅图像没有被删除？
**A:**
- 可能是图像文件仍然存在
- 检查图像源文件路径
- 尝试重新加载脚本后再运行

### Q5: 我需要选择要清理的对象吗？
**A:**
- **不需要**，脚本会自动扫描整个图纸
- 只需输入命令即可

### Q6: 可以只清理某一类型的对象吗？
**A:**
- 目前脚本一次清理所有类型
- 如需选择性清理，可手动编辑脚本
