# 🧹 CLEANDWG 图纸深度清理大师 - 插件使用说明

**CLEANDWG** 是一款专为 AutoCAD 打造的高效 LISP 插件。只需一键操作，即可在后台静默、彻底地清理图纸中隐藏的各类“垃圾”和冗余数据，并最终输出一份详尽的清理数据统计报告。非常适合用于**图纸瘦身、图纸交付前的清理规范化**。

---

## ✨ 核心功能与清理范围

本插件会自动全局检索并处理以下 6 大类无效内容：

1. 🖼️ **光栅图像/照片**：全局删除并彻底从数据字典中拆离所有贴图、Logo 和照片。
2. 📊 **OLE 嵌入对象**：彻底清除图纸中内嵌的 Excel 表格、Word 文档或剪贴板图片等 OLE 实体。
3. 📄 **参考底图**：删除所有残留的 PDF、DWF、DGN 参考底图。
4. 🔗 **外部参照 (XREF)**：强行断开关联，自动拆离图纸内存在的所有外部参照文件。
5. 🗑️ **冗余块与空图层**：执行 3 次深度静默 `Purge`，清除图纸中未使用的废弃块参照、空图层、废弃标注样式及线型。
6. ✂️ **重复与重叠线条**：后台静默调用 CAD 的 `OVERKILL` 功能，自动消除重叠在一起的线段、圆弧，合并共线对象。

---

## 🚀 安装与加载方法

### 方法一：临时加载（最简单，重启 CAD 后失效）
1. 打开 AutoCAD 和需要清理的图纸。
2. 将 `CleanCAD.lsp` 文件直接**拖拽**到 AutoCAD 的绘图区空白处。
3. 命令行提示 `>>>[图纸清理插件] 加载成功！` 即表示准备就绪。

### 方法二：常规加载（推荐）
1. 在 CAD 命令行输入 `APPLOAD` (或简写 `AP`) 并回车。
2. 在弹出的窗口中，找到并选中 `CleanCAD.lsp` 文件，点击**加载**。

### 方法三：永久自动加载（一劳永逸）
1. 输入 `APPLOAD` 打开加载对话框。
2. 点击右下角 **“启动组” (Startup Suite)** 下方的 **“内容...”** 按钮。
3. 点击 **“添加”**，将 `CleanCAD.lsp` 加入列表中，关闭窗口。以后每次打开 CAD 都会自动加载该插件。

## 💡 使用教程

1. 确保插件已加载。
2. 在 CAD 命令行输入快捷命令：
   ```text
   CLEANDWG
   ```
3. 按下回车键，程序将在后台静默运行（由于彻底屏蔽了系统冗余提示，运行过程中不会产生烦人的弹窗）。
4. 运行结束后，按 `F2` 打开命令行历史记录，您将看到类似如下的**精确清理汇总清单**：

```text
=============================================
            图纸深度清理已完成！             
---------------------------------------------
 [清理汇总清单]                              
  - 光栅图像/照片  : 成功删除 5 个
  - OLE 嵌入对象   : 成功删除 2 个
  - PDF等参考底图  : 成功删除 0 个
  - XREF 外部参照  : 成功拆离 1 个
  - 冗余块/空图层  : 成功清理 234 个
  - 重复与重叠线条 : 成功消除 150 个
=============================================
```

## ❓ 常见问题解答 (FAQ)

### Q1: 提示“重复与重叠线条 : 已跳过 (当前CAD不支持此功能)”是什么意思？
**答：** 这意味着您当前的 AutoCAD 软件中缺失 `OVERKILL`（消除重叠线）原生命令。
*   **原因**：通常是因为安装 AutoCAD 时没有勾选安装 **“Express Tools (扩展工具)”**，或者您使用的是极其精简版的 CAD。
*   **影响**：插件会自动跳过这一步以防止程序卡死，其他 5 项清理工作依然会完美执行，不受影响。
*   **解决办法**：重新运行 CAD 安装程序，补充安装 Express Tools，或升级至更新版本的完整版 AutoCAD。

### Q2: 插件会把我图纸里正常使用的门窗、标注等“块”删掉吗？
**答：不会。** 插件对“块”的处理逻辑调用的是深度的 `Purge`（清理）命令。它只会清理那些**曾经导入过，但目前图纸画面中根本没有在使用**的废弃定义。您图面上能看到的实体块绝对安全。

### Q3: 为什么清理出来的冗余块/空图层数量特别大？
**答：** 这是正常现象。因为插件执行的是**嵌套清理**（循环执行了3次）。一个废弃的块内部可能嵌套了废弃的线型和图层，插件会将它们像剥洋葱一样彻底清算，所以统计出来的字典词条数量往往比预想的要多，这也证明了图纸得到了真正的“瘦身”。

**⚠️ 免责声明：** 
由于本插件具有不可逆的全局删除与重组功能（如打断重复线段、拆离所有外部参照），建议在执行 `CLEANDWG` 命令前，**先保存一次图纸原文件**，以防万一。
